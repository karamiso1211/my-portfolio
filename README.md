## 🛠 Tech Stack
このサイトで使用している技術スタックです。

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## ✨ Features
- **レスポンシブ対応:** スマホ・PCどちらでも快適に閲覧できます。
- **高速なページ遷移:** Next.jsの機能により、読み込み待ち時間の少ない体験を実現しています。
- **実績管理:** 制作物をリスト形式で管理し、詳細ページへ誘導します。
